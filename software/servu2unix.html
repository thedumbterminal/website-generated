Servu2unix - Overview

Converts users from the the popular Windows Serv-U FTP server to UNIX system accounts

Features

  • Creates passwd and shadow files
  • Sets GID and UID from given values

Requirements

  • Linux operating system
  • Perl installed

Servu2unix - Help

Run the following command for help. 

./servu2unix
