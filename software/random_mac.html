random_mac - Overview

This is a simple bash script that will randomise the MAC address of the specified interface.

Requirements

  • BASH installed
  • The following shell utilities
    • ip (from iproute)
  • Root level access

random_mac - Help

Run the following command for help. 

./random_mac.sh
