Correct BlackBerry TCP options for use with T-Mobile

If you want to access the net from your BlackBerry using an application other than the T-Zones browser you will need to set the following options.

In Settings -> Options -> Advanced Options -> TCP

APN: general.t-mobile.uk
Username for APN: user
Password for APN: one2one

The settings will be applied without the need for a reboot.
