How to open directories in the same window with Gnome

By default Gnome's Nautilus file management application will open directories in a new window, this is called the Spatial mode. This can be annoying once you have many windows open.

To change this behavior so the same window is used you will need to switch to the browser mode as follows:

  1. Open Nautilus
  2. Go to Edit menu then select Preferences.
  3. Then the click on the Behavior tab.
  4. Check the Always open in browser windows.
  5. Close Nautilus and re-open.
