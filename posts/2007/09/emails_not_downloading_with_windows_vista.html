Emails not downloading with Windows Vista

If you can't download emails using third party mail clients such as Thunderbird in Windows Vista, run the following command via a command prompt as administrator.

First select start menu->accessories->command prompt.

Then right click run as administrator

Next paste in:

netsh interface tcp set global autotuninglevel=disabled
