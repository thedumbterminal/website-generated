Car PC 2007 - Software

The Car PC uses a standard Slackware 12 minimal install with the following changes:

  • Custom kernel with minimal drivers
  • mpg123 installed
  • init scripts altered to cope with readonly root fs
  • Autofs configured
  • Alsa configured
  • BASH script to generate playist and start mp3 player

You can download the complete OS here:

  • carpc_os.tar.gz (~50Mb)
